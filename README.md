# Pera Enerji Takvimi

Astroloji ve manevi gelişim uygulaması — React Native / Expo ile geliştirilmektedir.

## Özellikler
- 🌙 Kişisel Doğum Haritası (Natal Chart)
- 📅 Astrolojik Takvim (2026 önemli olaylar)
- 📿 Zikirmatik (13+ esma, kalıcı kayıt)
- 🏠 Günlük Enerji Paneli

## Teknoloji
- **Framework:** React Native (Expo)
- **Navigasyon:** Expo Router
- **State:** Zustand + AsyncStorage
- **Backend:** Supabase
- **Stil:** NativeWind (TailwindCSS)

## Geliştirme

```bash
npm install
npm start
```

> Expo Go ile veya simülatörde çalıştırın.

## Ortam Değişkenleri

`.env` dosyası oluşturun (örnek için `.env.example` bakın):

```
EXPO_PUBLIC_SUPABASE_URL=...
EXPO_PUBLIC_SUPABASE_ANON_KEY=...
```
