# 🐛 PET Bug Tracker

> Bu dosya, uygulamadaki tüm tespit edilen hataların merkezi takip noktasıdır.
> Yeni hata bulunduğunda `/report-bug` komutuyla otomatik eklenir.
> Hata çözüldüğünde `/fix-bug` komutuyla güncellenir.

## Durum Açıklamaları
- 🔧 **Açık** — Henüz çözülmedi
- 🔄 **İnceleniyor** — Üzerinde çalışılıyor
- ✅ **Çözüldü** — Fix uygulandı ve test edildi
- 🚀 **Deployed** — TestFlight'a yüklendi

## Öncelik Seviyeleri
- 🔴 **Kritik** — Uygulama çöküyor veya veri kaybı
- 🟠 **Yüksek** — Önemli özellik çalışmıyor
- 🟡 **Orta** — Küçük işlev bozukluğu
- 🟢 **Düşük** — Kozmetik / UX iyileştirmesi
